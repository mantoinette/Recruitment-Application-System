# RecruitPro Frontend

React + Vite single-page application for the [RecruitPro recruitment system](../README.md).

## Quick start

```bash
npm install
npm run dev
```

See the [root README](../README.md) for full setup instructions, demo accounts, and API configuration.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server at http://localhost:5173 |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview production build at http://localhost:4173 |
| `npm run lint` | Run ESLint |

## Backend

The API must be running at **http://localhost:8080**. Configure the base URL in `src/api/axios.js` if needed.
